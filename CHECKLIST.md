## 🎯 КОНЕЧНАЯ ПРОВЕРКА

### Полный путь пользователя

1. [ ] Пользователь пишет боту `/start`
2. [ ] Видит описание канала и кнопку "Купить подписку"
3. [ ] Нажимает кнопку
4. [ ] Получает ссылку на оплату Robokassa
5. [ ] Оплачивает подписку
6. [ ] Получает уведомление об успехе
7. [ ] Получает ссылку для входа в канал
8. [ ] Входит в приватный канал
9. [ ] Может прочитать контент

### За 3 дня до окончания

- [ ] Бот отправляет напоминание
- [ ] Пользователь может продлить подписку
- [ ] Продление добавляет 30 дней

### После окончания подписки

- [ ] Статус меняется на "expired"
- [ ] Пользователю отправляется уведомление
- [ ] Доступ к каналу закрывается (опционально)
